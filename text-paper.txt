ABSTRACT

Communicated by Y. Lai

Document classification systems become more and more complex with the need to deal with new document formats or categories while obtaining a low error rate when classifying more and more documents. Such systems
need to have important features including (1) the ability to eliminate ambiguity to improve precision or reduce
the error rate, (2) the capability to detect and reject documents belonging to new categories or new variations.
Previous studies often focused on closed datasets or solely on the problem of novelty detection, without evaluating the ability to reject ambiguous results after the novelty detection. In this paper, we propose an end-to-end
document classification algorithm including both novelty and ambiguity rejection. The proposed algorithm utilizes deep metric learning to compact the knowledge space, and then uses the last hidden layer‚Äôs features as
input for an unsupervised KNN-based method for novelty and ambiguity rejection. Extensive experiments and
analysis on private and public benchmark datasets demonstrate the effectiveness of our proposed algorithm. The
algorithm provides the capability to handle new documents while effectively rejecting ambiguity to enhance the
precision, recall of known categories, and coverage rate of the end-to-end document classification system.

1. Introduction
Documents are used every day for supporting different administrative tasks, for instance in the financial and administrative domains.
Document analysis and understanding remain a challenge as they involve the need to read and understand the textual content and understand its spatial organization. As reading a document is a complex task,
a first step in recent systems consists of classifying documents in order
to detect their class/objective and then define the process to be used for
the final task. This document classification step requires high accuracy
because any error will lead to a waste of time and money, and finally call
for a human operator to manually reject the documents or correct the
mistake. Document classification faces two main difficulties which are,
on the one hand, the large number of document categories (more than
200 document categories are generally used in industrial systems) and
on the other hand, the evolution of these document categories as new
and unseen formats of documents are generated every day. For example,
in financial applications, invoices, purchase orders, or bank statements
may have different templates, field locations, and formats depending on
the company or software used. Even the format of the same category of

‚ąó

document from the same company may change over time. These difficulties make it very challenging to develop a system with high accuracy
that is also able to detect new categories or formats.
Powered by parallel computing capabilities and big data, deep learning networks have achieved significant performance in various fields
from NLP [1,2] to Computer Vision (CV) [3], leading to great impact on
academic research and commercial applications. The algorithms are also
used in document analysis systems [4,5], especially for document classification [6‚Äď8]. In the studies conducted so far, classifiers are trained on
a dataset consisting of several categories and evaluated on a test dataset
containing unseen documents belonging to the same set of categories.
However, in reality, there are novel/unseen documents that do not belong to these categories nor have novel formats of existing categories
(see Fig. 1). In this case, the classifiers need to categorize them as novel
documents or an unknown category. Most novelty detection studies have
been in the field of computer vision [9‚Äď11]. Although there are studies [12] on novelty detection for document images, the number is still
limited.

2.2. Novelty detection
Novelty detection is a critical task for deploying machine learning
models in the real-world applications. We also call to it as out-ofdistribution (OOD) detection where a given sample is detected as novelty
(or OOD) if it is relatively far away from known in-distribution (ID)
samples [18‚Äď20]. Depending on the accessibility of OOD data, novelty
detection methods are categorized into three kinds: supervised [21], selfsupervised [22], and unsupervised [9,18] approaches. In the supervised
approach, through training on both ID and OOD data, models are expected to generalize information from both ID and OOD data, thereby
being able to recognize OOD samples [21]. However, this approach faces
the challenge of collecting a sufficiently large dataset that covers all possible OOD cases in reality, which is infeasible. This is not only because
OOD data is highly diverse, but also because in the case of administrative documents, new data categories continuously emerge whenever
new services are introduced. On the other hand, in order to generalize effectively on the ID data and exhibit distinctions from OOD data,
the self-supervised methods [22] employ augmentation techniques (e.g.,
image rotation) on the ID dataset to enable the model to learn a feature
space in which the inter-class separation is greater than the intra-class
separation. However, this method may be challenging to apply in the
context of natural language due to the limitations of text augmentation
techniques. Unsupervised methods only utilize ID data during training [11,18,23]. Initially, a classifier is trained on the ID dataset. Then,
this classifier is employed to extract features. A scoring function transforms these features of a given sample into a real number. By using the ID
data, an appropriate threshold is chosen based on the desired true positive rate. The OOD detector classifies the input as OOD if the score is
smaller than the selected threshold value. The extracted features can be
the class probabilities [24‚Äď26] or other latent space metrics [9,10,27].
Recent studies on novelty detection in pattern recognition have achieved
state-of-the-art efficacy, often focusing on utilizing embedding features
and Euclidean distance [9,11,18].
Despite being extensively studied in the field of computer vision,
out-of-distribution detection has not received much attention in the
field of natural language processing and document analysis. Shu et al.,
2017 [25] build a multi-class classifier with a 1-vs-rest final layer of
sigmoid rather than softmax then use Gaussian fitting to evaluate thresholds for OOD Detection. Cho et al. [28] enhance OOD detection in NLP
by leveraging intermediate representations and show the potential of intermediate representations in OOD detection. Wu et al. [29] and Zhou
et al. [30] improve OOD detection performance by using contrastive
learning. Gunel et al. [31] apply Supervised Contrastive Learning for
Pre-trained Language Models to enhance OOD detection. Most recently,
Larson et al. [12] in 2022, have introduced a novel dataset (namely
RVL-CDIP-O) for evaluating the out-of-distribution performance of document classification. The authors conducted experiments to compare
the effectiveness of novelty detection of different network architectures
combined with OOD detection methods on this dataset after training
the network on the popular RVL-CDIP dataset. Their highest AUC and
FPR@TPR95 were 92.30 % and 46.10 %, respectively.

2.3. Deep metric learning
Overall, the Deep Metric Learning (namely DML) [32,33] is a family of methods that focus on optimizing neural networks based on the
principle of enhancing the similarity or dissimilarity between semantic
features of samples depending on whether the samples belong to the
same object or different objects. DML was conceived based on the idea
of a contrastive loss [34‚Äď36]. Bromley et al. [37] proposed the first contrastive loss function for Siamese Networks in 1993. The loss function
measures the similarity between every pair of samples, where positive
pairs belong to the same category, and negative pairs belong to different categories. This loss function was later formally defined by Chopra
et al. [38] with application to Face Verification in 2005, where positive
pairs belong to the same person. Similarly, the Triplet loss was proposed
in studies by Schroff et al. [39], using triplets: an anchor, a positive
example, and a negative example. By using Triplet Loss, the distance
between an anchor and a positive example, both of which belong to
the same identity, is minimized, while the distance between the anchor
and a negative example of a different identity is maximized. Building on
that, many studies have focused on applying contrastive learning to both
self-supervised and supervised learning. In self-supervised contrastive
learning [36], where data lacks labels, a positive pair often involves data
augmentations of the same sample, and negative pairs are constructed
using the anchor and randomly selected samples from the mini-batch.
It has also been utilized and has achieved high effectiveness in supervised learning with a labeled dataset [32,35,40], where positive samples
belong to the same label, and negative samples belong to different labels.
Recently, DML has been employed in numerous studies [9,18,26] to
optimize neural networks, thereby enhancing the effectiveness of the
novelty detection and Out-of-distribution detection problems. Cevikalp
et al. [9] employed a supervised center contrastive loss function to train
their neural networks, followed by utilizing the Euclidean distance to
the class centers as a score function for novelty detection. This approach achieved state-of-the-art accuracies on the CIFAR-10, MNIST,
and Fashion-MNIST datasets. Yang et al. [26] designed a novel onevs-all loss to optimize their networks, followed by utilizing embedding
feature distance-based rejection and probability-based rejection to improve novelty detection performance. Sun et al. [11] utilized supervised
contrastive learning to optimize a CNN and subsequently combined
it with K-nearest neighbors to improve the performance of Out-ofdistribution detection on several image benchmark datasets. Zhou et al.
[30] employed margin-based contrastive learning to optimize neural
network then used the Mahalanobis distance in the model‚Äôs penultimate
layer to detect out-of-distribution instances in Sentiment Analysis, Topic
Classification, and Question Classification. However, there has not yet
been any research assessing the effectiveness of DML for the problem of
novelty and ambiguity rejection.

Margin-based contrastive loss function as a deep metric learning method
to optimize neural networks and compress knowledge. This is the first
study exploring the combination of text-based classifiers and KNN algorithm for not only novelty detection but also ambiguity rejection. Our

4.5. Discussion
This study has thoroughly evaluated a comprehensive algorithm for
real-world document classification problem. The research has conducted
a thorough assessment of numerous out-of-distribution methods and loss
functions across three scenarios using both private and public datasets.
The research findings indicate that the contrastive loss function effectively compacts knowledge during training, consequently enhancing
the performance of out-of-distribution detection. The last hidden layer‚Äôs
features contain more valuable information than logits of the output
layer for novelty detection problem. All embedding feature-based methods demonstrate superior novelty detection performance compared to
methods utilizing the logits information. Among them, the KNN-based
methods not only emerge as the best novelty detection algorithm but
also effectively eliminate ambiguous results, leading to a significant increase in PRE to nearly perfect levels, exceeding 99.50 % on the private
datasets and achieving an approximately 9 % improvement on the public RVL-CDIP dataset. This result indicates that the correlation between
a given document and its nearest neighbors from the training set provides valuable information for detecting novel documents and rejecting
ambiguous documents. In terms of end-to-end document classification,
KNN* stands out as the most efficient. KNN* removes ambiguous results when there is inconsistency between KNN output and logits output.
Therefore, it improves precision and recall of known categories.
When combined with Margin-based loss functions, the KNN-based
detection methods achieve the best performance in three scenarios. The

However, there are various aspects that could be further improved

threshold that is not optimized for each category. Overall, the categories are optimized similarly, and the variance and standard deviation

be more complex. Secondly, the margin-based contrastive loss function
could be further fine-tuned for better knowledge compactness, requiring
more time and GPU resources for a more detailed evaluation. Finally,
the proposed algorithm may be applied to different datasets and neural network architectures, and further experimentation is needed in the
future.
